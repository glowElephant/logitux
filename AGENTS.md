# logitux — Agent Guide

리눅스에서 로지텍 마우스 버튼을 마우스 도식을 보고 클릭만으로 매핑하는 사내 배포용 도구.
(이 파일은 도구 비종속 가이드 — Cursor / Copilot / Codex 등에서 사용. Claude Code 전용 내용은 CLAUDE.md 참조)

## 목표

사내 리눅스 사용자 누구나, 복잡한 CLI 도구 없이, 마우스 그림에서 버튼을 클릭해 키조합을 등록할 수 있게 한다.

## 아키텍처

- **GUI (PySide6/Qt)**: 마우스 SVG 도식 + 버튼 핫스팟/연결선/라벨. 버튼 클릭 → 키조합 등록.
- **백엔드 (Solaar/HID++)**: 매핑 엔진은 Solaar를 숨겨 활용. Solaar 룰 편집은 사용자에게 노출하지 않음.
- **도식 데이터**: 모델별 SVG + 버튼 좌표. MX Master 3/4부터, 데이터 추가로 모델 확장.
- **상시 실행**: MX 시리즈는 온보드 저장이 없어 변환 소프트웨어 상시 실행 필요 (autostart).

## 가드레일 (피할 것)

- Solaar 룰 직접 편집을 UI에 노출하지 말 것.
- MX 마우스 온보드 저장 가정 금지 (온보드는 G 게이밍 시리즈 전용).
- 마우스 모델 하드코딩 금지 — SVG+좌표 데이터로 외부화.
- 절대 경로 하드코딩 금지.
- X11 전용 가정 금지 — Wayland도 대상.
